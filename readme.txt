Make sure you have a valid URL (with video) copied to your clipboard or it'll just open and close.
If you have a valid URL it will attempt to download it, and should show an error if it fails...

Two options on how to run the script:

Auto Hot Key:

The hotkey for AHK is currently ctrl+d

If you want to change this, open the .ahk file and edit "^d"

# = WinKey/SuperKey
^ = Ctrl
! = Alt
+ = Shift

These can also be combined for a more complex combination
